Recupero de contraseña

Destino público para los enlaces de recuperación enviados por Supabase.

Usá este sitio como punto de aterrizaje cuando el usuario solicite recuperar su contraseña desde la app de escritorio.

Configurá tu app para usar un redirectTo público hacia: https://recupero-mim-qbfb.vercel.app/reset-password.html

Cómo funciona

  • El usuario pide recuperar contraseña en tu app.
  • Supabase envía el enlace a un dominio público.
  • El usuario abre la página y cambia su contraseña.

Instrucciones

  1. Reemplazá los valores en js/reset-password.js con tu Supabase URL y ANON KEY.
  2. Subí esta carpeta a un repo independiente.
  3. Habilitá GitHub Pages para el branch principal.
  4. Usá el enlace público en tu resetPasswordForEmail.
